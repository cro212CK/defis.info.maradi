# Site Web - Défis Informatique

Site de présentation pour le centre de formation "Défis Informatique" basé à Niamey, Niger.

## Structure des fichiers
- `index.html` - Structure principale du site
- `style.css` - Styles CSS du site
- `script.js` - Fonctionnalités JavaScript
- `images/` - Dossier pour les images (à créer)

## Fonctionnalités
1. Design responsive (mobile, tablette, desktop)
2. Menu de navigation mobile
3. Sections de présentation des formations
4. Formulaire de contact fonctionnel
5. Animations au défilement
6. Informations de contact du centre

## Technologies utilisées
- HTML5
- CSS3 (Flexbox, Grid, Variables CSS)
- JavaScript (ES6)
- Font Awesome (icônes)
- Google Fonts (Montserrat, Open Sans)

## Personnalisation
Pour personnaliser le site :
1. Modifiez les informations de contact dans `index.html`
2. Changez les couleurs dans les variables CSS dans `style.css`
3. Ajoutez vos propres images dans le dossier `images/`

## Déploiement
Le site peut être hébergé sur n'importe quel serveur web. Tous les fichiers nécessaires sont inclus.
